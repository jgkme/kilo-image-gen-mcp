# img-gen-mcp Skills

These skill files are bundled with the published `img-gen-mcp` package and are intended as complementary guidance for users of the MCP.

## Included skills

- `image-generation` - prompt structure and generation guidance for `generate_image` and `kilo_generate_image`
- `image-editing` - safe editing guidance for `edit_image`
- `background-removal` - local cutout and transparent PNG guidance
- `image-transform` - deterministic resize, crop, optimize, and finalize guidance
- `image-models` - provider, model, local backend, and tool discovery guidance
- `website-asset-workflow` - logos, favicons, hero images, and web delivery workflow

## How to use

- Treat these as a companion reference for the MCP tools.
- Use `website-asset-workflow` for any asset that needs to ship on a website or product surface.
- Use the narrower skills for focused tasks when you only need a single operation.
